รายการโครงการอบรมทั้งหมด

รวมคอร์สการฝึกอบรมจากศูนย์ฝึกวิชาชีพทางด้านวิทยาการคอมพิวเตอร์

อบรมการเขียนเว็บไซต์พื้นฐาน
เรียนรู้ HTML, CSS, JavaScript สำหรับผู้เริ่มต้นพัฒนาเว็บไซต์
อบรมการใช้โปรแกรม Excel เพื่อการวิเคราะห์ข้อมูล
รู้จักกับฟังก์ชันสำคัญ การจัดการข้อมูล และสร้างกราฟใน Excel
คอร์สพัฒนาแอปพลิเคชันมือถือด้วย Flutter
เรียนรู้การสร้าง Mobile App ทั้ง iOS และ Android ด้วยภาษา Dart
อบรมทักษะการออกแบบ UX/UI
เข้าใจพื้นฐานของการออกแบบประสบการณ์ผู้ใช้และอินเทอร์เฟซที่น่าใช้งาน
การวิเคราะห์ข้อมูลด้วย Python และ Pandas
ปูพื้นฐานการเขียน Python เพื่อวิเคราะห์และจัดการข้อมูล
พื้นฐานการสร้างเว็บไซต์ด้วย WordPress
เรียนรู้การใช้ WordPress เพื่อสร้างเว็บไซต์ได้ง่ายโดยไม่ต้องเขียนโค้ด
